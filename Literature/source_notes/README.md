# Source Notes

This folder contains short notes on the main sources used for the literature review. The goal is not to summarize every paper in full, but to document why each source was useful for the project.

Each note explains the source in relation to our routing model. In particular, the notes focus on what the source contributes to our understanding of vehicle routing, time windows, heterogeneous fleets, heuristics, scalability, uncertainty, or reproducibility.

The sources are not all used in the same way. Some support the mathematical problem class, some support the use of heuristic methods, others help explain limitations or newer research directions, while others are complementary sources. Technical documentation, such as solver manuals, is kept separate from this folder because it supports implementation rather than academic literature.

Together, these notes make the literature review easier to trace, they also help show that the project sources were selected because they connect to specific parts of the model, not only because they mention vehicle routing in general.
