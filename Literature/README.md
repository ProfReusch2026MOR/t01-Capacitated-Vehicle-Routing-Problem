# Literature

This folder contains literature, sources, and research notes for the T01 Capacitated Vehicle Routing Problem project.

## Purpose

The literature supports the theoretical and practical foundation of the project. It helps justify the model type, assumptions, constraints, and possible solution methods.

## Main Topics

- Vehicle Routing Problem
- Capacitated Vehicle Routing Problem
- Vehicle Routing Problem with Time Windows
- Time-dependent vehicle routing
- Heterogeneous fleet routing
- Logistics and retail distribution
- Heuristics and solver-based approaches

## Folder Contents

- `literature_summary.md` gives an overview of the most relevant literature areas.
- `sources.bib` contains structured BibTeX references.
- `source_notes/` contains short notes for individual sources.
- `papers/` may contain legally shareable PDFs or links to sources.
