# Archive

This folder contains older conceptual project files that are kept for transparency but are not part of the active final implementation.

The active final project is documented in the main repository folders:

* `README.md`
* `DATA/`
* `Docs/`
* `Results/`
* `Figures/`
* `Literature/`
* `src/`

The active implemented model is the 25-store Dr. Oetker → Lidl OWL routing case using Google OR-Tools, Clarke-Wright comparison, two vehicle capacity classes, time windows, service times, and demand scenarios S-99, S-152, and S-170.

Files in this archive may describe earlier theoretical or planning versions of the project. They should not be interpreted as the final executable model unless explicitly stated.

## Archived files

| File                                | Reason for archiving                                                                                                                                                                        |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Foundations_of_the_OR_Project.pdf` | Earlier conceptual framework with broader modelling ideas. It contains useful literature and future-extension material, but it does not fully match the final implemented code and results. |

## Important note

Archived files are preserved to show the development history of the project. The final model, final results, and reproducible implementation are defined by the current files in `src/`, `Results/`, `Docs/`, and the root `README.md`.
