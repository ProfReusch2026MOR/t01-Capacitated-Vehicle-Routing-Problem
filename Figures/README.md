# Figures

This folder contains visual material used in the project documentation and report.

For the traffic documentation task, the required file is:

`google_traffic_layer_bielefeld.png`

This screenshot shows the Bielefeld / OWL region with the Google Maps traffic overlay activated. It is used as visual support for the time-dependent traffic assumptions in the routing model.

The screenshot should not be interpreted as an exact historical traffic dataset. It is only used to show that traffic conditions can vary across the road network and that using time-dependent travel-time multipliers is a reasonable simplification for the project.

The traffic method itself is documented in:

`docs/traffic_data_method.md`
